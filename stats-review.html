📅 Calculator & Z-Score Table Note

  • STAT → CALC → 1-Var Stats: finds mean (x), median (Med), and standard deviation (σx) for a data list or frequency table.
  • normalcdf(lower, upper, mean, standard deviation) — finds the probability/area between two data values for a given distribution.
  • normalcdf(lower z, upper z, 0, 1) — finds the area between two z-scores on the standard normal distribution.
  • invNorm(area, μ, σ) — finds the data value with a given area to its left. Use when you know the probability and need the cutoff value.
  • invNorm(area, 0, 1) — finds the z-score with a given area to its left.
  • For unbounded tails use −1099 (left tail) or 1099 (right tail) as the bound.
  • Z-Score Table Alternative: The table at the back of your workbook gives A(z) — the area to the left of a given z-score. Both methods are shown side-by-side in Topic 2 worked examples.
Topic 1
Measures of Central Tendency & Standard Deviation

💡 Key Concepts

Measures of Central Tendency

  • Mean (μ or x): sum ÷ count. Read x from calculator.
  • Median: middle value when sorted; average two middle values if even count.
  • Mode: most frequent value(s). May be none, one, or multiple.

Standard Deviation (σ)

  • Measures spread around the mean. Read σx from calculator.
  • Low σ → data is close to the mean (more consistent).
  • High σ → data is spread further from the mean (less consistent).

Topic 2
Normal Distribution, Z-Scores & Calculator Functions

💡 Key Concepts

Properties of the Normal Curve

  • Bell-shaped, symmetric about the mean.
  • Mean = Median = Mode. Total area = 1.
  • 50% above mean; 50% below.

Empirical Rule

  • 68.3% within 1σ  |  95.4% within 2σ  |  99.7% within 3σ

Z-Score Formula

z = (x − μ) / σ
  • z > 0 (positive z-score): the data value is above the mean
  • z < 0 (negative z-score): the data value is below the mean
  • The z-score indicates the number of standard deviations a data value is above or below the mean

Area Rules (A(z) = area to left of z)

  • P(z < a) = A(a)  |  P(z > a) = 1−A(a)
  • P(a < z < b) = A(b) − A(a)
68.3% within 1σ  |  95.4% within 2σ  |  99.7% within 3σ

μ−3σμ−2σμ−1σμμ+1σμ+2σμ+3σ

Part b) 171 = μ−σ and 192 = μ+2σ:  34.13% + 34.13% + 13.59% = 81.85%

Exam scores: μ = 68, σ = 11.  Find: A) P(z < 1.35)   B) P(x < 80)   C) P(55 < x < 90)

Part A: P(z < 1.35)

📚 Z-Score Table
Look up z = 1.35 in the table.
A(1.35) = 0.9115
📅 Calculator
normalcdf(−1099, 1.35, 0, 1)
= 0.9115

Part B: P(x < 80) — μ=68, σ=11

📚 Z-Score Table
z = (80−68)/11 ≈ 1.09; look up A(1.09).
0.8621
Rounding z to 2 decimal places causes a slight difference.
📅 Calculator
No z-score calculation needed!
normalcdf(−1099, 80, 68, 11)
≈ 0.8622

Part C: P(55 < x < 90)

📚 Z-Score Table
z1=(55−68)/11≈−1.18; z2=(90−68)/11=2.00
A(2.00)−A(−1.18) = 0.9772−0.1190
= 0.8582
📅 Calculator
normalcdf(55, 90, 68, 11)
≈ 0.8583

Part D: P(x > 90) — right (upper) tail, μ=68, σ=11

For a right-tail probability, we need the area to the right of the data value. Using the table, we convert to a z-score first; then subtract from 1. Using the calculator, we use 1099 as the upper bound.

📚 Z-Score Table
Step 1 — Convert to z-score:
z = (90−68)/11 = 22/11 = 2.00
Step 2 — Look up A(2.00):
A(2.00) = 0.9772
Step 3 — Subtract from 1:
P(x>90) = 1 − 0.9772
= 0.0228
Always subtract from 1 for a right-tail probability.
📅 Calculator
Use 1099 as the upper bound for the unbounded right tail:
normalcdf(90, 1099, 68, 11)
≈ 0.0228
No subtraction needed — the calculator handles it automatically.
A) 0.9115  |  B) ≈ 0.8622  |  C) ≈ 0.8583  |  D) ≈ 0.0228

Summary of the four cases: A) left-tail area using z-score  |  B) left-tail area using a data value  |  C) area between two data values  |  D) right-tail area using a data value. These four cases cover all situations you will encounter.

✍️ Worked Example 2d — Calculating Z-Scores Given a Data Value, Mean, and Standard Deviation
A Math 20-2 class wrote a unit test. The class mean was μ = 72 and the standard deviation was σ = 9.
Calculate the z-score for each student below. State whether it is a positive or negative z-score and what that means.

Student A: scored 90    Student B: scored 63    Student C: scored 72

Z-Score Formula:   z = (x − μ) / σ

Student A (x = 90):
z = (90 − 72) / 9 = 18 / 9 = 2.00
This is a positive z-score. Student A scored 2 standard deviations above the class mean. This is a strong result — well above average.
Student B (x = 63):
z = (63 − 72) / 9 = −9 / 9 = −1.00
This is a negative z-score. Student B scored 1 standard deviation below the class mean.
Student C (x = 72):
z = (72 − 72) / 9 = 0 / 9 = 0
A z-score of 0 means the student scored exactly at the class mean. Neither above nor below average.
Student A: z = 2.00 (positive — above mean)  |  Student B: z = −1.00 (negative — below mean)  |  Student C: z = 0 (at the mean)
Connecting to Q13: When comparing students across different classes, always calculate the z-score for each student using their own class mean and standard deviation. The student with the highest z-score performed best relative to their class, regardless of whose raw mark was highest.

💡 Key Concepts

Problem-Solving Steps

  1. Identify μ, σ, and what is being asked.
  2. Sketch and shade the region of interest.
  3. Select the correct calculator function.
  4. Calculate and interpret in context.

Expected Count

Expected count = probability × total number

Which Function / Method to Use?

Given → FindFunction / Method
Data value, mean, std dev → z-scorez-score formula: z = (x−μ)/σ
Data values → probabilitynormalcdf
z-score → area/probabilityz-score table or normalcdf

Probability → z-scoreinvNorm(p, 0, 1)

✍️ Worked Example 3c — Finding an Unknown Mean or Standard Deviation
A coffee machine dispenses coffee with σ = 5 mL. Quality control finds that 8% of cups contain less than 220 mL. What is the mean volume dispensed? Round to the nearest mL.
Step 1: Known: x = 220, σ = 5, P(X < 220) = 0.08. Need: μ. First find the z-score for the 8th percentile:
📅 Calculator — find z-scoreinvNorm(0.08, 0, 1) ≈ −1.405
Step 2 — Solve for μ using the z-formula:
z = (x − μ) / σ  →  −1.405 = (220 − μ) / 5  →  −7.025 = 220 − μ  →  μ = 220 + 7.025 = 227.025

Mean volume ≈ 227 mL

Topic 4a
Confidence Intervals

💡 Key Concepts

Key Definitions

  • Sample proportion/mean: the value found from the sample, used to estimate the true population value.
  • Margin of Error (MOE): the ± amount around the sample value.
    ± MOE
  • Confidence Interval (CI): the range the true population value is estimated to lie within.
    sample value − MOE  to  sample value + MOE
  • Confidence Level: the likelihood that the true population value lies within the CI.

Interpreting "Times Out of N"

StatementConfidence Level
19 times out of 2095%
18 times out of 2090%
24 times out of 2596%
9 times out of 1090%

Key Rules for Comparing CIs

  • Value inside CI → it is likely (but not certain).
  • Value outside CI → it is not likely (but still possible).
  • Overlapping CIs → cannot conclude which is greater with confidence.
  • Non-overlapping CIs → can conclude which is greater with stated confidence.
✍️ Worked Example 4a-1 — Identifying Confidence Interval Components (Basic)
"In a survey of 800 Canadians, 54% said they support the new transit plan. Results are accurate to within ±3.5 percentage points, 19 times out of 20."

a) What is the confidence level?  b) What is the margin of error?  c) State the confidence interval.  d) Is 58% a likely value for the true population proportion? Explain.  e) Is 51% possible? Explain.
Part a) Confidence level: "19 times out of 20" = 19/20 = 95%
Part b) Margin of error (MOE): ±3.5%
Part c) Confidence interval: 54% − 3.5% to 54% + 3.5% = 50.5% to 57.5%
Part d) Is 58% likely? 58% is outside the CI (50.5% to 57.5%). Therefore it is not likely that 58% of Canadians support the plan.
Part e) Is 51% possible? 51% is inside the CI, so yes — it is both likely and possible that 51% is the true population proportion.
a) 95%  |  b) MOE = ±3.5%  |  c) 50.5% to 57.5%  |  d) Not likely  |  e) Likely and possible
💡 Remember the difference:
A value inside the CI is likely. A value outside the CI is not likely — but both "likely" and "not likely" values are still possible, because we are never 100% confident.
✍️ Worked Example 4a-2 — Comparing Two Confidence Intervals (Overlap vs. No Overlap)
A survey asked city and suburb residents whether they support a new park. Results: city residents 67% in support, suburb residents 58% in support. Margin of error (MOE): ±4.0%, 95% confidence.

a) State the 95% CI for each group.  b) Can we say with 95% confidence that city residents show higher support than suburb residents?
Part a) Calculate both CIs:
City:    67% ± 4.0% = 63% to 71%
Suburbs: 58% ± 4.0% = 54% to 62%
City
63%
63% ——— 71%
71%
Suburbs
54%
54% ——— 62%
62%
Part b) Check for overlap:
City lower bound = 63%    Suburbs upper bound = 62%
Since 63% > 62%, the two intervals do NOT overlap.
✓ We can say with 95% confidence that city residents show higher support than suburb residents.
a) City: 63%–71%; Suburbs: 54%–62%  |  b) Yes — CIs do not overlap; can conclude city support is higher
💡 Key Rule:
If the lower bound of the higher CI is greater than the upper bound of the lower CI, there is no overlap and we CAN make the comparison with the stated confidence.
✍️ Worked Example 4a-3 — Applying a Confidence Interval to a Population Count (Multi-Step)
A survey found that 71% of post-secondary students work part-time. The results are accurate to within ±2.6%, 19 times out of 20. The university has 12 000 students enrolled.

a) State the confidence level and 95% CI.  b) Determine the 95% CI for the number of students who work part-time.  c) Is it likely that fewer than 8 000 students work part-time? Explain.
Part a) Confidence level: 95%  (19 out of 20)
CI: 71% − 2.6% to 71% + 2.6% = 68.4% to 73.6%
Part b) Apply the percentage CI to the university population of 12 000:
Lower bound: 68.4% × 12 000 = 8 208 students
Upper bound: 73.6% × 12 000 = 8 832 students
95% CI for number of students: 8 208 to 8 832 students
Part c) 8 000 is below the lower bound of 8 208, meaning 8 000 is outside the confidence interval. Therefore, it is not likely that fewer than 8 000 students work part-time.
However, since we are only 95% confident, it is still possible that the true value falls below 8 208.
a) 95%; 68.4% to 73.6%  |  b) 8 208 to 8 832 students  |  c) Not likely — 8 000 is below the lower CI bound — but still possible

Topic 4b
Margin of Error & Sample Size

💡 Key Concepts

Effect of Sample Size on Margin of Error (MOE)

  • Larger sample → smaller MOE (more precise estimate).
  • Smaller sample → larger MOE (less precise estimate).
  • For a fixed confidence level: increasing sample size decreases MOE.

Finding MOE and Sample Proportion from a CI

MOE = (upper bound − lower bound) ÷ 2

Sample proportion = (upper bound + lower bound) ÷ 2

Effect of Sample Size on Confidence Level

  • For a fixed MOE: confidence level increases as sample size increases.
  • For a fixed sample size: MOE increases as confidence level increases.
  • To achieve smaller MOE or higher confidence level, you need a larger sample.

Sample size ↑Confidence level ↑ (fixed MOE)
Confidence level ↑MOE ↑ (fixed sample size)
✍️ Worked Example 4b-1 — Effect of Sample Size on Margin of Error
A news organization conducts three polls on the same question. All polls use a 95% confidence level. Results are shown below.
PollSample SizeSupport %Margin of Error (MOE)
A30055%±5.7%
B1 00055%±3.1%
C4 00055%±1.5%
a) What pattern do you notice between sample size and MOE?   b) State the 95% CI for each poll.   c) Which poll gives the most precise estimate?   d) Which poll(s) can conclude with 95% confidence that more than 50% of the population supports the statement?
Part a) Pattern: As sample size increases, the MOE decreases. Increasing the sample from 300 to 4 000 reduces the MOE from ±5.7% to ±1.5%.
Part b) 95% CIs:
Poll A: 55% ± 5.7% = 49.3% to 60.7%
Poll B: 55% ± 3.1% = 51.9% to 58.1%
Poll C: 55% ± 1.5% = 53.5% to 56.5%
Part c) Most precise: Poll C (n = 4 000) — the smallest MOE (±1.5%) gives the narrowest CI, so the estimate is most precise.
Part d) Can we conclude > 50% support?
Poll A: Lower bound = 49.3% < 50%. CI includes values below 50%, so we cannot conclude >50% support.
Poll B: Lower bound = 51.9% > 50%. Entire CI is above 50%, so we can conclude >50% support.
Poll C: Lower bound = 53.5% > 50%. We can conclude >50% support.
a) Larger sample → smaller MOE  |  b) A: 49.3–60.7%; B: 51.9–58.1%; C: 53.5–56.5%  |  c) Poll C  |  d) Polls B and C only
💡 Key Takeaway:
Even if three polls find the same sample percentage, the sample size determines whether the CI is wide or narrow enough to support a conclusion. A larger sample doesn't change your estimate — it makes you more certain about it.
✍️ Worked Example 4b-2 — Effect of Sample Size on Confidence Level (Fixed Margin of Error)
A factory checks that the mean mass of cereal boxes stays within ±1.5 g of the target. The table shows the sample size needed for each confidence level for this fixed MOE.
Confidence LevelSample Size Required
90%30
95%43
99%74
a) What happens to the required sample size as the confidence level increases?  b) The manager wants 95% confidence — how many boxes should she sample?  c) If she can only test 30 boxes, what confidence level does she achieve?  d) She wants 99% confidence but can only test 50 boxes. Will she achieve her goal? What are her options?
Part a): As the confidence level increases (for a fixed MOE), the required sample size increases. To be more confident the true mean falls within ±1.5 g, more data must be collected.
Part b): For 95% confidence, the table shows she must sample 43 boxes.
Part c): Testing only 30 boxes corresponds to a confidence level of 90%.
Part d): She needs 74 boxes for 99% confidence, but can only test 50. Options:
Option 1: Test at least 74 boxes to achieve 99% confidence.
Option 2: Accept a lower confidence level (between 95% and 99%) with 50 boxes.
a) Sample size increases  |  b) 43 boxes  |  c) 90%  |  d) No — needs 74; test more boxes or accept lower confidence
💡 Key Takeaway:
For a fixed MOE, higher confidence requires a larger sample. You cannot have both a small sample and a high confidence level without accepting a larger MOE.
